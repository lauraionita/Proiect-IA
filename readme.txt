O sa trebuiasca sa instalati requests:
https://pypi.python.org/pypi/requests

in cmd: pip install requests

De asemenea, trebuie instalata libraria pt transformare docx in txt
https://pypi.python.org/pypi/docx2txt

in cmd: pip install docx2txt

Varianta simpla e pentru potrivire cu ce s-a facut la modulul 2 si 3.
Varianta cu NP este cea completa.

Documentele trebuie sa fie tip docx.
